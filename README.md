# NoteApp

This is a Simple but Complete Android Note App with clean architecture!

### Here i used:
- MVVM Architecture
- Room Database
- ViewBinding
- One Activity- Multiples Fragments
- Navigation Components
- Action with Animation
- Search View
- DeffUtil & Data Binding with  RecyclerView
